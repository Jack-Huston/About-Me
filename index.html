ASEN/CSCI 5264, Decision Making under Uncertainty

Planning and control with hidden state and partial observability
  • Model decisions under uncertainty using probabilistic state and observation models.
  • Plan with MDPs using value iteration and policy iteration.
  • Apply reinforcement learning methods to learn policies from experience.
  • Plan with POMDPs using belief updates, including Bayesian and particle filtering concepts.
  • Cover multi-agent decision making through game theory and Markov games.
